# CineMind 다음 기능 추천 목록

## 1. '태그' 기능 도입 (코멘트 기능의 대안 및 확장)

코멘트 기능이 기술적인 문제에 부딪혔으므로, 더 간단하고 구조화된 방식인 '태그' 기능을 도입하여 '나만의 생각 기록'이라는 목표를 다른 방식으로 달성하는 것을 제안합니다.

*   **기능 설명:**
    *   사용자가 영화 상세 정보 모달에서 자신만의 태그를 자유롭게 추가할 수 있습니다. (예: `#인생영화`, `#주말순삭`, `#N차관람`, `#OST맛집`)
    *   사용자는 자신이 추가한 태그를 클릭하여, 해당 태그가 달린 모든 영화를 모아볼 수 있습니다.
    *   프로필 페이지에 '나의 태그 클라우드'를 시각적으로 표시하여, 자신의 취향을 한눈에 파악할 수 있게 합니다.

*   **기대 효과:**
    *   자유로운 텍스트 입력보다 데이터 관리가 용이하여, 이번에 겪었던 것과 같은 복잡한 버그를 피할 수 있습니다.
    *   단순한 기록을 넘어, 태그를 통해 자신만의 영화를 '분류'하고 '필터링'하는 강력한 개인화 기능을 제공합니다.
    *   '나만의 시네마 저널' 컨셉을 세련되고 재미있는 방식으로 구현할 수 있습니다.

*   **구현 계획 (간략히):**
    1.  **DB:** `user_movie_tags` 라는 새로운 테이블 생성 (`user_id`, `movie_id`, `tag_name`).
    2.  **백엔드:** 태그 추가/삭제/조회 API 구현.
    3.  **프론트엔드:** 영화 모달에 태그 입력 및 표시 UI 추가, 프로필에 태그 클라우드 UI 추가, 태그별 영화 목록 페이지 신규 생성.

## 2. '컬렉션' 기능 추가

사용자가 특정 주제에 맞춰 자신만의 영화 목록을 만들고 관리하는 기능입니다. Letterboxd, 왓챠피디아 등 많은 영화 앱의 핵심 기능 중 하나입니다.

*   **기능 설명:**
    *   사용자는 '내 인생 최고의 SF 영화 TOP 10', '비 오는 날 보고 싶은 영화' 등 자신만의 주제로 '컬렉션'을 생성할 수 있습니다.
    *   각 컬렉션에 영화를 추가하고, 순서를 정하고, 간단한 설명을 덧붙일 수 있습니다.
    *   (추후 확장) 다른 사용자가 만든 컬렉션을 구독하거나 '좋아요'를 누를 수 있습니다.

*   **기대 효과:**
    *   단순히 영화를 평가하는 것을 넘어, 영화를 '큐레이션'하는 즐거움을 제공하여 앱 사용성을 크게 높입니다.
    *   사용자가 앱에 더 많은 시간과 애정을 쏟게 만드는 강력한 동기 부여가 됩니다.

*   **구현 계획 (간략히):**
    1.  **DB:** `collections` 테이블과 `collection_movies` 테이블 생성.
    2.  **백엔드:** 컬렉션 생성/수정/삭제 및 영화 추가/제거 API 구현.
    3.  **프론트엔드:** 프로필에 '내 컬렉션' 탭 추가, 컬렉션 생성 및 편집 UI, 컬렉션 상세 페이지 신규 생성.

## 3. '비슷한 영화' 추천 기능

영화 상세 정보 모달 하단에, 현재 보고 있는 영화와 비슷한 다른 영화들을 추천해주는 기능입니다.

*   **기능 설명:**
    *   TMDB API는 특정 영화와 비슷한 영화 목록을 추천해주는 기능을 이미 제공하고 있습니다.
    *   영화 상세 모달에서 이 API를 호출하여, 비슷한 영화들의 포스터 목록을 가로 스크롤 형태로 보여줍니다.

*   **기대 효과:**
    *   사용자가 자연스럽게 다른 영화를 탐색하게 유도하여 앱에 더 오래 머무르게 합니다.
    *   구현 난이도가 비교적 낮으면서도 사용자 만족도를 크게 높일 수 있는 '가성비' 좋은 기능입니다.

*   **구현 계획 (간략히):**
    1.  **백엔드:** TMDB의 'Similar Movies' API를 호출하는 새로운 API 엔드포인트 추가.
    2.  **프론트엔드:** `MovieModal.tsx` 내부에 비슷한 영화 목록을 표시하는 `MovieCarousel`과 같은 컴포넌트 추가.
