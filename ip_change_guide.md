### **CineMind 앱 로컬 네트워크 연결 가이드 (IP 주소 변경 방법)**

개발 중인 모바일 앱(클라이언트)이 내 컴퓨터에서 실행 중인 서버(백엔드)와 통신하려면, 서버의 "집 주소"를 정확히 알고 있어야 합니다. 이 집 주소가 바로 **로컬 IP 주소**입니다.

집, 학교, 카페 등 다른 Wi-Fi에 연결할 때마다 이 IP 주소는 바뀌기 때문에, 네트워크를 변경할 때마다 아래의 과정을 진행해 주셔야 합니다.

#### **1단계: 내 컴퓨터의 현재 IP 주소 확인하기**

먼저 터미널(Terminal) 앱을 열고, 다음 명령어를 입력하여 현재 사용 중인 네트워크 정보를 확인합니다.

```bash
ifconfig
```

명령어를 실행하면 여러 네트워크 인터페이스(`lo0`, `en0`, `awdl0` 등)의 정보가 길게 나옵니다. 우리가 찾아야 할 것은 **현재 활성화된 유선 또는 무선랜의 IP 주소**입니다.

**찾는 요령:**

1.  목록에서 `en0` 라는 항목을 찾습니다. (Mac에서 Wi-Fi에 연결된 경우 대부분 `en0`입니다.)
2.  `en0` 항목 아래에 `status: active` 라고 되어 있는지 확인합니다.
3.  바로 그 아랫줄 근처에 있는 `inet` 으로 시작하는 줄을 찾습니다. 이것이 현재 컴퓨터의 로컬 IP 주소입니다.

**예시:**

```
en0: flags=8863<UP,BROADCAST,SMART,RUNNING,SIMPLEX,MULTICAST> mtu 1500
    ...
    inet 192.168.201.137 netmask 0xffffe000 broadcast 192.168.223.255
    ...
    status: active
```

위 예시에서 현재 IP 주소는 `192.168.201.137` 입니다. (`127.0.0.1`은 항상 자기 자신을 가리키는 주소이므로 제외합니다.)

#### **2단계: 클라이언트(앱) 설정 파일 수정하기**

이제 앱에게 "서버의 새 주소는 여기야!"라고 알려줄 차례입니다.

1.  Visual Studio Code와 같은 코드 에디터에서 CineMind 프로젝트를 엽니다.
2.  왼쪽 파일 탐색기에서 다음 경로의 파일을 엽니다.
    `cinemind-client/constants/config.ts`

3.  파일을 열면 아래와 같은 코드가 보입니다.

    ```typescript
    // constants/config.ts
    const API_BASE_URL = 'http://192.168.?.?:8000'; // 이전에 사용하던 IP 주소

    export default API_BASE_URL;
    ```

4.  위 코드에서 `'http://` 와 `:8000'` 부분은 그대로 두고, **가운데의 IP 주소 부분만** 1단계에서 찾은 새로운 IP 주소로 교체합니다.

    **수정 예시:** (만약 새로 찾은 IP가 `192.168.201.137`이라면)

    ```typescript
    // constants/config.ts
    const API_BASE_URL = 'http://192.168.201.137:8000'; // 새로 찾은 IP 주소로 변경

    export default API_BASE_URL;
    ```

5.  파일을 저장합니다.

#### **3단계: 모든 서버 재시작하기 (가장 중요!)**

변경된 설정을 적용하려면, 실행 중이던 모든 서버를 완전히 껐다가 다시 켜야 합니다.

1.  **백엔드 서버 재시작:**
    *   백엔드 서버가 실행 중이던 터미널 창에서 `Control` + `C` 키를 눌러 서버를 중지합니다.
    *   다시 다음 명령어를 입력하여 서버를 실행합니다.
        ```bash
        uvicorn main:app --host 0.0.0.0 --port 8000
        ```

2.  **프론트엔드 서버 재시작:**
    *   Expo(클라이언트)가 실행 중이던 터미널 창에서 `Control` + `C` 키를 눌러 서버를 중지합니다.
    *   다시 다음 명령어를 입력하여 서버를 실행합니다.
        ```bash
        npm start
        ```
